## 🛠️ Security Toolset & Technologies

The following security-oriented frameworks, utilities, and targets were used throughout this program:

* **Attacker Machine:** Kali Linux VM (Debian-based)
* **Target Machines:** Metasploitable2 (Linux), DVWA (Damn Vulnerable Web Application)
* **Scanning & Recon:** Nmap, Nessus Essentials, Whois, Nslookup, Shodan
* **Web Hacking & Interception:** Burp Suite (Community Edition), Firefox Developer Tools
* **Packet & Traffic Analysis:** Wireshark, Netcat, hping3
* **Exploitation Frameworks:** Metasploit, Hydra (Brute-forcing), John the Ripper (Hash Cracking)
* **Defensive Utilities:** OpenSSL, iptables, Linux System Auditing, Apache/Nginx Web Server config
